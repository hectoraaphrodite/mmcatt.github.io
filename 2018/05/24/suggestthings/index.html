一些很棒的资料，分享 & 推荐

发表于 2018-05-24 | 更新于 2018-07-11 | 分类于 something

有时候会看到一些很好的资料。选出我认为最有用的部分，放到这里，与各位分享之。

  1. My Personal Source Code: Books to Learn Analysis
    作者在第一段这么说道：

    偶尔人们来找我,说他们想学到某些东西。他们会问“我怎么成为一个好的测试人员”或“我如何设计测试用例”或者“我如何自动化”具体。不过,这些都是不正确的问题。更好的问题,它解决了所有其他的,是“我如何成为一个称职的分析师?“分析是所有技术工作的根源。这是主几乎一切的关键。你几乎将自动成为一个好的测试人员,测试用例设计,或者自动化的无论你选择,如果你掌握分析。(是的,同等优先的还有其他因素,比如人性,节制,和超然。今天,我将重点分析。)

因此这是关于思考，分析问题的推荐书单。作者是从事软件测试行业的，所以也是重点推荐给同行业的小伙伴。当然，这些都是方法论的书，从事技术工作的小伙伴都值得阅读。

  1. JOE COLANTONIO: Automation awesomeness
    这是一个网站，正如其名字一样，是关于自动化测试的。网站上有很多有趣的访谈。当然，你得有足够好的英文听力。或者抛弃VOA吧，干脆用这个来练听力？
    而创办这个网站的人也很有趣，他曾经的梦想是成为一名著名的吉他手，还在伯克利音乐学院学习过。某一天他意识到摇滚乐的生活方式或许并不适合他，于是，BAM！——软件测试自动化工程师诞生了。

  2. Test Automation Frameworks
    这个算是电子书啦，没有纸质版，没有中文版。最早看到别人推荐是在一个公众号上，推荐的自动化测试的书籍。

  3. Leet Code
    题库。可以刷一下算法题、数据库题，还有一些别的题。总有朋友问我要这个网站的地址，于是也放在这里了。

这篇，会持续更新下去，请保持关注。
